# GPT-Clone

GPT-Clone is a web application that allows users to interact with an AI chat model using Express.js and Hugging Face's inference API. This project provides a simple chat interface where users can send messages and receive responses generated by an AI model.

## Features

- Interactive chat interface
- AI responses powered by Hugging Face's inference API
- Simple and clean design using Tailwind CSS
- EJS templating for dynamic content rendering

## Prerequisites

- Node.js and npm installed on your machine
- A Hugging Face API key
- Basic understanding of Express.js, Node.js, and EJS

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/gpt-clone.git
   cd gpt-clone
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**

   Create a `.env` file in the root directory and add your Hugging Face API key and the port number:

   ```env
   HUGGINGFACE_API_KEY=your_huggingface_api_key
   PORT=3000
   ```

4. **Start the server:**

   ```bash
   npm start
   ```

5. **Open your browser and go to `http://localhost:3000` to start using the chat interface.**

## Usage

- **Home Route (`GET /`):** Renders the main chat interface.
- **Chat Route (`POST /chat`):** Handles user input, sends it to the AI model, and displays the response.

## Code Overview

- `index.js`: Main server file that sets up the Express application, routes, and integrates with Hugging Face's API.
- `views/index.ejs`: EJS template file for rendering the chat interface.
- `public/`: Static files such as CSS, JavaScript, or images (if any).
